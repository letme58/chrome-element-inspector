# Chrome 元素查看器插件

一个用于自动查看元素 XPath、CSS 选择器的 Chrome 浏览器插件，支持彩色轨迹功能。

## ✨ 功能特点

- 🔍 智能识别页面元素
- 📋 自动生成多种 XPath 定位方式
- 🎯 快速分析元素属性
- 📝 一键复制元素信息
- 🌈 彩色鼠标轨迹动画

## 🎮 快捷键

| 快捷键 | 功能 |
|--------|------|
| **Z** | 开启/关闭元素检查模式 |
| **X** | 固定/取消固定当前选中的元素 |
| **C** | 显示/隐藏元素信息提示框 |
| **T** | 开启/关闭彩色轨迹 |

## 📦 安装方法

1. 下载本仓库代码
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目文件夹

## 🚀 使用方法

1. 点击插件图标，打开弹出窗口
2. 点击「开始检查」按钮（或按 **Z** 键）
3. 移动鼠标到目标元素上查看信息
4. 点击可复制的内容即可复制到剪贴板

```

```
